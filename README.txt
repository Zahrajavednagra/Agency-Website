Project Title: BrightWeb Agency Website

Website Purpose:
This website is created for a premium digital agency to showcase company information, services, portfolio work, and provide a contact option for potential clients.

Features Implemented:
- Multi-page website (Home, About, Services, Contact)
- Responsive glassmorphism navigation menu
- Luxury hero section
- Modern service cards
- Image slider using JavaScript
- Contact form with validation
- Newsletter subscription form
- Embedded Google Map
- Portfolio section
- Modal popup for project preview
- PHP backend contact form
- Responsive design for mobile, tablet, and desktop
- Hover effects, gradients, glow buttons, and animations

Tools Used:
- HTML
- CSS
- JavaScript
- PHP
- VS Code

How to Run:
1. Open the project folder in VS Code
2. Run index.html using Live Server for frontend
3. Use XAMPP localhost to test PHP form
4. Navigate through all pages using the menu
5. Test responsiveness using browser inspect tool

Backend Integration:
A PHP file (contact.php) is included to handle contact form submission.
